손안의 여행 리뉴얼

한국관광공사 2024 여행업계 디지털전환사업(DX) 참여 - "손안의 여행" 리뉴얼 - (퍼블리싱 100%)

Responsive web / PHP, gnuboard, 영카트 기반

URL : https://www.sonantravel.com/

(주)손안의여행은 여행자들이 안전하고 쉬운 여행을 즐길 수 있도록 인솔자와 함께하는 여행과 여행컨설팅을 제공하는 자유배낭여행 전문여행사입니다. (주)손안의여행 사이트는 한국관광공사 여행사 디지털 전환사업에 걸맞게 여행 예약 시스템 뿐만 아니라 인솔자와의 매칭, 문의나 후기 게시판, 여행 설명회 접수 등의 고객과의 소통창구를 마련했습니다. 또 카드 형태의 레이아웃과 여행 예약 현황 제공으로 주목도와 편의성을 높였습니다.

루벤티스 R&D 사업 참여

루벤티스 - 비즈오토메이트 업무 자동화 서비스 페이지 구축 - (디자인 100% / 퍼블리싱 100%)

Responsive Web / JSP 기반

URL : http://bizautomate.co.kr/

루벤티스㈜는 다양한 산업군을 지원하는 통합 물류 솔루션과 물류 특화된 빅데이터/AI 기술력을 보유한 기업입니다.
기업의 기술 개발 관련 비용을 지원하는 정부 R&D 지원사업을 통하여 루벤티스의 업무 자동화 서비스 "비즈오토메이트"
소개 및 구독 페이지를 디자인과 퍼블리싱 모두 진행했습니다.
회계, 물류 관련 위주의 이미지와 아이콘으로 전문성을 강조했으며, 메인 화면은 지원 서비스 분야 4가지로 심플하게
구성했습니다. 각 서비스의 페이지는 구독신청, 이용 가이드, 문의 컨텐츠로 구성했습니다.

템플릿 홈페이지 제작 서비스 "웹비서"

오픈컴 템플릿 사이트 제작 대행 서비스 "웹비서" 구축 - (디자인 100% / 퍼블리싱 100%)

Responsive web / PHP, gnuboard, 영카트 기반

URL : https://webiseo.kr/

웹비서는 오픈컴에서 제작한 템플릿을 바탕으로 고객이 원하는 홈페이지를 빠르게 제작하는 서비스입니다.
웹비서는 타사 서비스와는 다르게 템플릿 판매 후 고객이 직접 만들지 않고, 일대일 CS 상담원을 배정하여 요청사항에 따라 내부 작업자가 직접 홈페이지를 제작하여 고객은 비즈니스에만 집중할 수 있게 도와줍니다.
사이트 디자인부터 영카트를 이용한 쇼핑몰 퍼블리싱, 판매하는 템플릿 상품 대부분까지 디자인 및 퍼블리싱을 진행했습니다.
캐릭터는 우주를 배경으로 한 사이트 디자인과 어울리도록 제작했으며, 작은 넥타이 포인트는 웹비서의 신뢰를 상징합니다.

Responsive web / PHP, gnuboard, 영카트 기반

가온 3D는 ㈜라온비아이엠에서 운영하는 건설장비 모델링 자료 판매 쇼핑몰이며, 제 작업 중 최초로 그누보드 영카트를 이용하여 작업을 진행했습니다. 기존 영카트 레이아웃에는 없는 제품 목록 썸네일에 슬라이드를 레이아웃을 적용하여 제품의 3D 모델링을 바로
다각도로 확인할 수 있도록 하고, 제품 상세 페이지에는 태그를 적용하여 제품 정보를 빠르게 확인할 수 있도록 했습니다.

Responsive web / Wordpress
